# 🐱 猫咪情绪识别数据采集详细方案

> **用途**：供家人协同执行的数据采集指南
> **目标**：为5种猫咪情绪状态采集图片和音频数据，用于训练AI识别模型
> **采集对象**：可乐、草草（两只猫）

---

## 📋 一、采集目标总览

### 5种情绪状态

| 编号 | 状态名称 | 图标 | 说明 |
| :--- | :--- | :--- | :--- |
| ① | 饥饿乞食 | 🍚 | 猫咪需要食物 |
| ② | 警惕警告 | ⚠️ | 猫咪感到威胁不安 |
| ③ | 心情愉悦 | 😊 | 猫咪感到舒适满足 |
| ④ | 无聊求玩 | 🎾 | 猫咪需要娱乐 |
| ⑤ | 标记领地 | 🏠 | 猫咪在宣示主权 |

### 数据总量

| 数据类型 | 每种情绪 | 每只猫 | 总量 |
| :--- | :--- | :--- | :--- |
| 图片（视频关键帧） | 40-50张 | 20-25张 | 200-250张 |
| 音频 | 30-40段 | 15-20段 | 150-200段 |

> **重要原则**：两只猫的数据量要**大致相等**，保证数据均衡。

## 🎬 二、视频录制方案（用于提取图片关键帧）

### 2.1 录制设备与设置

**设备**：手机（推荐后置摄像头）

**设置建议**：
- 分辨率：1080P（1920×1080）
- 帧率：30fps
- 格式：MP4
- 横屏拍摄（ landscape ）

### 2.2 录制要求

| 要求 | 标准 | 说明 |
| :--- | :--- | :--- |
| 时长 | 10-30秒/段 | 捕捉完整行为序列 |
| 距离 | 1-3米 | 能看清猫咪全身 |
| 光线 | 明亮均匀 | 自然光最佳，避免逆光 |
| 稳定性 | 尽量手持稳定 | 可靠墙或扶物 |
| 背景 | 简洁 | 避免杂乱背景 |
| 角度 | 多角度拍摄 | 正面、侧面、俯视、平视 |

### 2.3 各情绪录制场景指南

#### ① 饥饿乞食 🍚

**最佳录制时机**：
- 饭前（猫咪饿了的时候）
- 主人在厨房准备食物时
- 早上起床后、傍晚喂食前

**捕捉的行为**：
- 猫咪走向食盆
- 站在食盆旁看着主人叫
- 蹭主人的腿
- 尾巴竖起，尾巴尖微弯
- 舔嘴巴

**录制技巧**：
- 提前把手机准备好，等待猫咪自然发生
- 从猫咪开始走向食盆时录，到吃到食物前停止

---

#### ② 警惕警告 ⚠️

**最佳录制时机**：
- 陌生人来访时
- 听到异常声音（敲门声、雷声）时
- 有其他动物靠近时
- 猫咪看到窗外有猫时

**捕捉的行为**：
- 飞机耳（耳朵向后贴）
- 瞳孔放大
- 身体压低或弓起
- 尾巴炸毛或猛甩
- 低沉的呜呜声、哈气声

**录制技巧**：
- ⚠️ 安全第一：不要为了拍摄而激怒猫咪
- 保持距离，用变焦拉近
- 等待自然发生，不要人为制造威胁

---

#### ③ 心情愉悦 😊

**最佳录制时机**：
- 主人抚摸时
- 阳光下晒太阳时
- 刚吃饱满足时
- 睡觉前放松时
- 猫咪主动求摸摸时

**捕捉的行为**：
- 眼睛半闭或缓慢眨眼
- 身体放松，侧躺或蜷缩
- 尾巴轻轻摆动
- 咕噜声（呼噜声）
- 面部表情平和
- 主人抚摸时闭眼享受

**录制技巧**：
- 这是比较容易录制的状态
- 可以主动创造场景（抚摸、晒太阳）
- 多录一些，因为猫咪日常经常处于这种状态

---

#### ④ 无聊求玩 🎾

**最佳录制时机**：
- 傍晚活跃时段（猫的活跃期）
- 看到玩具时
- 主人回家时
- 深夜跑酷时

**捕捉的行为**：
- 叼着玩具走向主人
- 扒门或扒柜子
- 在主人脚边转圈
- 追逐玩具或光点
- 眼睛有神，身体活跃
- 短促兴奋的叫声

**录制技巧**：
- 准备好逗猫棒、激光笔等玩具
- 可以引导猫咪玩耍，但要记录自然行为
- 动作可能较快，注意对焦

---

#### ⑤ 标记领地 🏠

**最佳录制时机**：
- 猫咪日常活动时
- 新物品出现时（新家具、新箱子）
- 清洁后的家具（气味被消除）
- 早晨和傍晚（标记行为高发期）

**捕捉的行为**：
- 用头蹭家具或墙角
- 爪子抓挠沙发或猫抓板
- 身体站立，前爪交替抓挠
- 尾巴竖起颤抖
- 脸颊蹭物品

**录制技巧**：
- 耐心等待，标记行为会自然发生
- 可以在猫咪常去的地方观察
- 不要打扰猫咪的标记行为

## 🎵 三、音频录制方案

### 3.1 录制设备与设置

**设备**：手机录音功能

**设置建议**：
- 格式：WAV（无损）或 M4A
- 采样率：44100 Hz
- 声道：单声道
- 距离：1-2米

### 3.2 录制要求

| 要求 | 标准 | 说明 |
| :--- | :--- | :--- |
| 时长 | 3-5秒/段 | 每段包含1-3次叫声 |
| 环境 | 安静室内 | 关闭电视、音乐、窗户 |
| 背景音 | 无 | 无人声、无其他动物声 |
| 音量 | 适中 | 不要太小（听不清）也不要爆音 |

### 3.3 各情绪音频特征

| 情绪 | 声音特征 | 录制时机 |
| :--- | :--- | :--- |
| 饥饿乞食 | 短促的"喵"声（频率较高）、连续多次叫、声音急切 | 饭前 |
| 警惕警告 | 低沉的"呜呜"声、哈气声（嘶嘶声）、警告性低吼 | 感到威胁时 |
| 心情愉悦 | 呼噜声（持续低频）、轻柔的短喵、咕噜声 | 抚摸、晒太阳时 |
| 无聊求玩 | 中等长度的喵叫、叼着玩具时的呜呜声、短促兴奋叫声 | 傍晚活跃时 |
| 标记领地 | 可能有低沉咕噜声、通常较安静、抓挠摩擦声（可选） | 日常标记时 |

### 3.4 音频录制技巧

1. **提前准备**：关闭所有可能产生噪音的设备（电视、风扇、空调外机等）
2. **距离适中**：1-2米，太远录不清，太近可能打扰猫咪
3. **保持安静**：录制时不要说话，不要移动
4. **多次录制**：同一次叫声可以多录几段，后期选最好的
5. **及时检查**：每录完一段，立即回放检查质量

### 3.5 音频剪辑

**工具推荐**：Audacity（免费开源，电脑端）

**剪辑步骤**：
1. 打开录音文件
2. 找到猫咪叫声清晰的片段
3. 裁剪成3-5秒的短片段
4. 适当调整音量（标准化）
5. 导出为 WAV 格式

## 🖼️ 四、关键帧提取方案

### 4.1 提取工具

**推荐工具**：FFmpeg（命令行，高效）或 VLC播放器（图形界面，简单）

### 4.2 方法A：VLC播放器截图（简单，推荐家人使用）

**操作步骤**：
1. 用VLC播放器打开视频
2. 播放视频，找到猫咪行为最典型的画面
3. 暂停视频
4. 点击菜单：视频 → 拍照（或快捷键 Shift+S）
5. 截图自动保存到"图片"文件夹

**每段视频截图数量**：3-5张

**选帧原则**：
- ✅ 选择猫咪姿态最典型的瞬间
- ✅ 选择画面最清晰的帧
- ✅ 选择猫咪完整可见的帧
- ❌ 避免猫咪模糊、运动中的帧
- ❌ 避免猫咪被遮挡的帧

### 4.3 方法B：FFmpeg批量提取（高效，适合电脑熟练者）

**安装FFmpeg**：
```bash
winget install Gyan.FFmpeg
```

**从视频中每隔2秒提取一帧**：
```bash
ffmpeg -i input_video.mp4 -vf fps=0.5 output/frame_%04d.jpg
```

**提取指定时间段的帧（第5秒到第15秒，每秒1帧）**：
```bash
ffmpeg -i input_video.mp4 -ss 5 -to 15 -vf fps=1 output/frame_%04d.jpg
```

**提取后需要人工筛选**：从批量提取的图片中，挑选最清晰、最典型的3-5张。

### 5.5 本地标注方案（图片 + 音频）

#### 标注方法：文件夹分类法（推荐）

**原理**：把图片/音频按情绪状态放在不同的文件夹里，文件夹名就是标签，EasyDL会自动识别。

**步骤**：
1. 在 `数据集/03_处理图片/` 下创建5个文件夹：
   ```
   03_处理图片/
   ├── 01_饥饿乞食/
   ├── 02_警惕警告/
   ├── 03_心情愉悦/
   ├── 04_无聊求玩/
   └── 05_标记领地/
   ```
2. 在 `数据集/04_处理音频/` 下同样创建5个文件夹：
   ```
   04_处理音频/
   ├── 01_饥饿乞食/
   ├── 02_警惕警告/
   ├── 03_心情愉悦/
   ├── 04_无聊求玩/
   └── 05_标记领地/
   ```
3. 将处理好的图片和音频拖入对应文件夹
4. 检查无误后打包上传

#### EasyDL上传步骤

**图片分类项目**：
1. 登录百度EasyDL → 创建项目 → 选择"图像分类"
2. 数据管理 → 导入数据 → 选择"文件夹分类"
3. 将 `03_处理图片/` 打包成zip文件上传
4. EasyDL自动识别文件夹名为标签

**音频分类项目**：
1. 登录百度EasyDL → 创建项目 → 选择"语音分类"
2. 数据管理 → 导入数据 → 选择"文件夹分类"
3. 将 `04_处理音频/` 打包成zip文件上传
4. EasyDL自动识别文件夹名为标签

#### 标注注意事项
- ✅ 每个文件夹的数据量尽量接近（差异不超过20%）
- ✅ 标注前先检查质量，不合格的不要放入
- ✅ 可乐和草草的数据可以混在同一文件夹（统一模型训练）
- ✅ 保持文件名规范，方便追溯来源

## ✅ 八、质量检查清单

### 8.1 视频质量检查

每段视频录制后检查：
- [ ] 视频清晰，猫咪可见
- [ ] 时长在10-30秒之间
- [ ] 光线明亮，无逆光
- [ ] 猫咪行为符合该情绪状态
- [ ] 背景相对简洁
- [ ] 无剧烈晃动

### 8.2 音频质量检查

每段音频录制后检查：
- [ ] 猫咪叫声清晰可辨
- [ ] 时长在3-5秒之间
- [ ] 背景安静，无明显噪音
- [ ] 音量适中，无爆音
- [ ] 无其他人声干扰

### 8.3 关键帧质量检查

每张关键帧提取后检查：
- [ ] 图片清晰，不模糊
- [ ] 猫咪完整可见（包括尾巴）
- [ ] 猫咪姿态符合该情绪状态
- [ ] 光线明亮均匀
- [ ] 猫咪是画面主体
- [ ] 无遮挡

### 8.4 数据均衡检查

最终检查：
- [ ] 每种情绪至少40张图片（可乐+草草）
- [ ] 每种情绪至少30段音频（可乐+草草）
- [ ] 可乐和草草的数据量大致相等（差异不超过20%）
- [ ] 每种情绪的图片涵盖多角度、多距离、多场景
- [ ] 所有文件命名规范
- [ ] 所有文件分类正确

## ⚠️ 十、注意事项

### 10.1 猫咪福利优先

- ❌ **不要为了拍摄而强迫猫咪**
- ❌ **不要在猫咪生病、紧张时强行拍摄**
- ❌ **不要制造威胁来拍摄"警惕警告"**
- ✅ **耐心等待猫咪自然展现行为**
- ✅ **如果猫咪不愿意，立即停止**

### 10.2 数据真实性

- ✅ **记录自然行为**，不要摆拍
- ✅ **多场景、多时段**采集，保证多样性
- ✅ **如实标注**，不确定的不采
- ❌ **不要用同一张图片重复使用**
- ❌ **不要从网上下载图片**（用自己家的猫）

### 10.3 数据安全

- ✅ **定期备份**到电脑和云盘
- ✅ **原始数据保留**，不要删除视频
- ✅ **处理后数据单独存放**
